# vim new plugins notes

## tcomment

    gc{motion} 
    gcc          :: Toggle comment for the current line
    gC{motion}   :: Comment region
In visual mode:
    gc           :: Toggle comments
    gC           :: Comment selected text

    <c-_><c-_>   :: :TComment

    <Leader>__       :: :TComment
